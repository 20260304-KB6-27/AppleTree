# <a href="https://20260304-kb7-27.github.io/AppleTree/html/SplashScreenPage.html">🐣 KB 루키 - 지점장 키우기</a>

**KB 루키 - 지점장 키우기**는
CS 지식 퀴즈를 풀며 캐릭터를 성장시키는 게이미피케이션 기반 학습 서비스입니다.

사용자는 다양한 유형의 CS 퀴즈를 풀어 점수와 경험치를 얻고
캐릭터를 성장시켜 KB 국민은행 지점장까지 승진시키는 것을 목표로 합니다.

단순한 문제 풀이가 아니라 게임처럼 캐릭터를 키우며 학습하는 경험을 제공합니다.

<br/>

# 🎯 서비스 목표

- CS 지식을 재미있게 학습

- 퀴즈 + 캐릭터 성장 시스템을 통한 몰입도 향상

- 게임 요소를 통한 학습 지속성 강화

<br/>

# 👤 캐릭터 성장 시스템

퀴즈에서 정답을 맞추면 점수와 경험치를 얻고 캐릭터가 성장합니다.

- 레벨업 시 경험치는 0으로 초기화됩니다.

## 캐릭터 성장 단계
### 🐣 1단계: IT’s Your Life 교육생

서비스 시작 시 기본 캐릭터입니다.

### 🏦 2단계: KB 국민은행 신입 행원

경험치 **300 달성** 시 성장

### 👔 3단계: KB 국민은행 지점장

경험치 **500 달성** 시 최종 성장

<br/>

# 🧠 퀴즈 시스템

사용자는 퀴즈를 풀어 점수와 경험치를 획득할 수 있습니다.

## 퀴즈 진행 방식

1. 메인 화면에서 퀴즈 시작 버튼을 클릭합니다.

2. 문제가 한 문제씩 화면에 표시됩니다.

3. 제한 시간 1분 동안 총 10문제를 풉니다.

4. 답을 선택하면 다음 문제로 자동 이동합니다.

5. 패스 버튼을 누르면 해당 문제는 자동 오답 처리됩니다.

6. 시간이 종료되면 최종 결과 화면이 표시됩니다.

## 퀴즈 유형
1. **4지선다**: 객관식 문제 중 하나를 선택합니다.

2. **O / X**: 문장의 참/거짓을 판단합니다.

3. **순서 맞추기**: 개념의 올바른 순서를 맞춥니다.

<br/>

# ⭐ 점수 시스템

- 문제 1개당 10점
- 한 게임에서 최대 100점 획득 가능
- 획득한 점수는 캐릭터 경험치로 반영

> 예시) 10문제 중 3문제 정답 → 30점

# 🎮 게임 플레이 방법

1. 시작하기 버튼 클릭

2. 1분 동안 10문제 퀴즈 풀기

3. 정답을 맞추면 점수에 따른 경험치 획득

4. 경험치를 모아 캐릭터 레벨업

5. KB 국민은행 지점장까지 성장
